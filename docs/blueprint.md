# **App Name**: Elite Escapes

## Core Features:

- Dynamic Page Structure Rendering: Renders all main page sections including the top bar, main header, hero section, various content grids, and footer, following the specified layout and aesthetic.
- Interactive Search & Filters: Implements the multi-tab floating search card within the hero section, featuring input fields for 'Where to?', 'When?', 'Flying from', 'Guests', and a 'Search' button, with responsive design.
- Content Display Grids: Presents featured holidays, popular destinations, and last-minute offers in adaptive grid layouts, where each card displays relevant details like images, pricing, and calls to action.
- Informational & Trust Widgets: Displays static informational elements such as trust badges, 'Why Book With Us' feature boxes (ATOL Protected, Award Winning, 24/7 Support), and customer statistics to build user confidence.
- Conversion-Focused Calls to Action: Implements key interactive buttons like 'Get a Quote', 'Call Us', 'View All Holidays', and 'View Deal', with specified styling to encourage user engagement.
- Newsletter Subscription Module: Provides an email input field and a subscribe button in the dedicated newsletter section for user sign-ups, implying backend data capture (even if mocked for MVP).
- AI-Enhanced Content Tool: Utilizes a generative AI tool to craft compelling and conversion-optimized headlines for holiday deals or the main hero subtitle, dynamically engaging potential customers.

## Style Guidelines:

- Primary color (branding elements, general text, non-CTA buttons): Rich, sophisticated dark teal (#125567), evoking trust and luxury. This color also forms the background of several key sections.
- Background color: A very light, desaturated teal-grey (#F2F5F6) for the overall light scheme of general content areas, providing a clean and subtle backdrop.
- Accent color (CTAs, key highlights): Vibrant, conversion-focused orange (#FB902B), creating clear and irresistible calls to action.
- Headline font: 'Poppins' (sans-serif), a geometric sans-serif that lends a modern, precise, and fashionable look suitable for headings. Body font: 'Inter' (sans-serif), chosen for its objective, neutral, and highly readable qualities, perfect for extended text.
- Use a consistent set of minimal, clean, and professional travel-themed icons. Ensure icons for trust badges (e.g., ATOL Protected) and feature boxes (e.g., Award Winning) are simple and recognizable.
- Utilize modern CSS Flexbox and Grid for responsive, pixel-balanced layouts. Apply consistent rounded corners (12px–16px) and soft shadows to cards and interactive elements for a premium feel. Maintain clean spacing throughout the design.
- Incorporate subtle animations such as a gentle shadow under the main header on scroll, and a sophisticated hover zoom effect on destination image cards to enhance interactivity without distraction.
